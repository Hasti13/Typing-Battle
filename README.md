# Typing Battle 🎮

Typing Battle is an interactive browser-based typing game developed using HTML, CSS, and JavaScript. The objective of the game is to type the falling words before they reach the bottom of the screen. As the player's score increases, the game gradually becomes faster, making it more challenging and engaging.

This project was created to practice front-end web development concepts and JavaScript DOM manipulation while building a fun and interactive application.

## Features

* Real-time score tracking
* Lives system
* Dynamic difficulty progression
* Animated falling words
* Restart functionality
* Simple and user-friendly interface
* Responsive design for different screen sizes

## Technologies Used

* **HTML** for the structure of the game
* **CSS** for styling and animations
* **JavaScript** for game logic and interactivity

## How to Play

1. Click the **Start** button to begin the game.
2. Type the words that appear on the screen.
3. Gain 10 points for each correctly typed word.
4. Lose one life whenever a word reaches the bottom.
5. The game ends when all lives are exhausted.
